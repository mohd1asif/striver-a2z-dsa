# DSA Practice (C++)

Solving problems from Striver's A2Z Sheet, topic by topic, in C++, as part
of my SDE prep.

## How this is organized

Each folder is one topic from the sheet, in order. Each file is one
problem, with the approach and time/space complexity written as a comment
at the top of the file.

## Rules I'm following

- Attempt every problem on my own before looking at any editorial.
- Once I understand the approach, I write the code myself — nothing here
  is copy-pasted.
- Solved problems get pushed the same day, not batched and dumped later.

## Why

Building a real foundation for SDE interviews, not just collecting solved
problems.
